Basics

Before you start to make your first webpage, there are some basic tags you should know. An html tag is how you will tell the computer what to dispaly on the webpage. All html tags start with < and end with >. With most tags there is an opening tag and a closing tag. For example, 

<h1>I'm a header.</h1>

There are three tags that will be on every webpage in these tutorials. They will all be included in every template you use on each tutorial, but you should know their meaning so you can use them when you make your own page from scratch. The first tag is the DOCTYPE tag. In defines the document as an html document. It should go on line 1 of any html file. There is no closing tag for the DOCTYPE tag. 

<!DOCTYPE html>

Next is the html tag. It holds all your html code. This should be the second and last line on your document. 

<html>All the code for my webpage...</html>

The final tag is the body tag. It holds the body of the webpage. You can place this pair of tags somewhere in the html tag, there are usully some other things above it but these don't matter yet. 

<body>The main body of my webpage</body>

If you combine all this you will get a basic html document that looks like this: 

<!DOCTYPE html>
  <html>
    <body>
      I will be a website.
    </body>
  </html>

You could use this code with one more line and you would see the text, "I will be a website" on the screen.
